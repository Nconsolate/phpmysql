### 问题及解决
* 将mysql部署到服务器。
        记得在控制台开端口。
* 本地运行连接服务器mysql正常。代码部署到服务器，连接mysql正常   ，视图不显现。  
    原因：服务器搭建的PHP环境为7.0，不支持mysql。
    解决：更换服务器PHP环境  <7.0即可，我这里更换的是PHP5.6环境。
